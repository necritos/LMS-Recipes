# Auth — Usuario final

Endpoints de autenticación para clientes de la plataforma.

**Prefijo:** `/api/v1/auth/`  
**JWT type:** `user`

| Endpoint | Descripción |
|----------|-------------|
| [register.md](./register.md) | Registro con email/contraseña |
| [login.md](./login.md) | Inicio de sesión |
| [refresh.md](./refresh.md) | Renovar access token |
| [logout.md](./logout.md) | Cerrar sesión (blacklist refresh) |
| [google.md](./google.md) | Login/registro con Google OAuth |
| [password-forgot.md](./password-forgot.md) | Solicitar código OTP |
| [password-verify-code.md](./password-verify-code.md) | Verificar código OTP |
| [password-reset.md](./password-reset.md) | Restablecer contraseña |
